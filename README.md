## 🌐 Развертывание

### GitHub Pages
1. Перейдите в Settings репозитория
2. Найдите раздел "Pages"
3. Выберите ветку "main" как источник
4. Сайт будет доступен по адресу: `https://ernurmahanbet00-jpg.github.io/Elsibo/`

### Кастомный домен
1. Выберите опцию "Custom domain" в GitHub Pages
2. Введите ваш домен
3. Обновите DNS записи у вашего провайдера:
   ```
   CNAME: your-domain.kz → ernurmahanbet00-jpg.github.io
   ```
4. GitHub автоматически создаст SSL сертификат
